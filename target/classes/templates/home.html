• O que foi utilizado nesse projeto:
Spring-Boot Spring Tool Suite Thymeleaf WebJars Hibernate Bootstrap 4
Spring Boot facilita a criação de aplicações autônomas e de produção que você pode apenas "executar" com um click. Adotamos uma visão simples da plataforma Spring e de bibliotecas de terceiros para que você possa começar com o mínimo de trabalho. A maioria dos aplicativos criados através do Spring Boot precisa de uma configuração de Spring muito pequena.
The Spring Tool Suite é um ambiente de desenvolvimento baseado em Eclipse que é customizado para desenvolver aplicações em Spring. Ele fornece um ambiente pronto para implementar, depurar, executar e implantar suas aplicações em Spring, incluindo integrações para o Server Pivotal Cloud Foundry, Git, Maven, AspectJ e muito mais.
Thymeleaf é um moderno mecanismo de templates do Java focado no lado do servidor para ambientes web e independente. Com módulos para o Spring Framework, uma série de integrações com suas ferramentas favoritas e a capacidade de conectar sua própria funcionalidade, o Thymeleaf é ideal para o desenvolvimento web de JVM de HTML5 moderno - embora haja muito mais que ele possa fazer.
WebJars são bibliotecas da web (por exemplo, jQuery; Bootstrap) empacotadas em arquivos JAR (Java Archive).
Hibernate ORM permite que os desenvolvedores escrevam mais facilmente aplicativos cujos dados sobrevivam ao longo do processo. Com uma estrutura Object / Relational Mapping (ORM), o Hibernate está preocupado com a persistência de dados, uma vez que se aplica à bancos de dados relacionais (via JDBC). Além de sua própria API "nativa", o Hibernate também é uma implementação da especificação Java Persistence API (JPA). Como tal, ele pode ser facilmente usado em qualquer ambiente com suporte em JPA, incluindo aplicativos Java SE, servidores de aplicação Java EE, Enterprise OSGi containers, etc
Bootstrap é uma ferramenta open source para desenvolvimento com HTML, CSS e JS que cria rapidamente protótipos de suas ideias ou constrói seus aplicativos com variáveis e 'mixins Sass', sistema de grades responsivas e extensos componentes pré-construídos e poderosos plugins construídos no jQuery.
